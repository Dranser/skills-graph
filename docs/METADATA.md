## Elevator Pitch

Skills Graph is a structured repository format for agent-facing skills. Instead of storing isolated prompt snippets, it organizes domains, routing rules, execution guidance, dependency expansion, and prioritized references into a reusable graph that coding and architecture agents can apply consistently.
